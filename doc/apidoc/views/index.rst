Views
=====

This section provides an overview of the views available in the `django_tinyuser` application.
Each view is designed to handle specific user interactions, such as inviting new users or displaying user profiles.
The views are implemented using Django's class-based views and are structured to support both standard HTTP requests
and HTMX requests for enhanced interactivity. Below is a detailed description of each view along with its
functionality and usage.
